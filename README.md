
# Samarth Portal

The landing page of the Samarth Portal for STPI


## Run Locally

Clone the project in Bash or Terminal

```bash
  git clone https://github.com/jatinxnewar/Samarth-Portal.git
```

Go to the project directory

```bash
  cd SAMARTH-WEBSITE
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
